# CLI Calculator

## Description

A CLI based calculator for unix/linux based systems, it allows the user to type entire equations into the program and the program will deal with all BODMAS logic and print the result, this program includes addition, subtraction, multiplication, division, power of and brackets functionality, it is made entirely with C++, and even has functionality to edit equations after writting them and by pressing the up arrow key you can see and use previous equations written into the program.

## Dependencies

- libreadline-dev

## Build
```bash
git clone https://github.com/Broang24/CLI-Calculator-Cpp.git
cd CLI-Calculator-Cpp
make
./calc
```
